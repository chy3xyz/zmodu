# zmodu v0.10.0

Code generation CLI for [ZigModu](https://github.com/chy3xyz/zigmodu). Modulith architecture scaffolding with AI-assisted development.

## Install

```bash
git clone https://github.com/chy3xyz/zmodu.git && cd zmodu && zig build
```

## Quick Start

```bash
zmodu scaffold --sql schema.sql --name myapp --json-style camel
cd myapp && zig build run
```

## Commands

```bash
zmodu scaffold --sql <file> --name <n>  # SQL → full project [primary command]
zmodu module <name>                      # empty module skeleton
zmodu orm --sql <file> [--out <dir>]     # ORM modules only (existing project)
zmodu api <name>                          # standalone API template
zmodu event <name>                        # event handler template
zmodu plugin stub --name <n>             # dependency stub for migration
zmodu plugin list                         # list stub status
zmodu life                                 # project evolutionary memory
```

## Scaffold Flags

```
--sql <file>       --name <name>        required
--out <dir>        --json-style camel   optional
--with-events      --with-auth          features
--with-resilience  --with-metrics
--with-transactions --with-redis
--with-websocket   --force --dry-run
```

## Generated Project Structure

```
src/
├── main.zig              # DB config + module wiring
├── shared/               # types, errors, response wrapper
├── business/root.zig     # cross-module orchestration
└── modules/<name>/
    ├── module.zig         # lifecycle + barrel re-exports
    ├── model.zig          # struct(sql_table_name) + json_names
    ├── persistence.zig    # ORM Repository(T)
    ├── service.zig        # CRUD + validate
    ├── api.zig            # REST handlers
    └── ext/               # AI extension (survives regeneration)
        ├── service.zig
        └── api.zig
```

## AI Contract

- **⛔ Generated files** (marked `@generated by zmodu`): NEVER edit
- **✅ AI writes**: `ext/`, `src/shared/`, `src/business/`, `src/compat/`
- **Update**: change schema.sql → re-run `zmodu scaffold`
- **Verify**: `zig build` must be 0 errors
